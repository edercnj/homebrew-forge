# homebrew-forge

Homebrew tap oficial do [Forge](https://github.com/edercnj/forge).

```bash
brew install edercnj/forge/forge
```

> A fórmula `Formula/forge.rb` é **gerada automaticamente** pela esteira de
> release do repositório principal (`scripts/update-homebrew-tap.sh`), a partir
> dos checksums de cada tarball publicado. Não edite à mão.
